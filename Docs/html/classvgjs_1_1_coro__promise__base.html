Detailed Description

Base class of coroutine Coro_promise, derived from Job_base so it can be scheduled.

The base promise class derives from Job_base so it can be scheduled on the job system! The base does not depend on any type information that is stored with the promise. It defines default behavior and the operators for allocating and deallocating memory for the promise.
