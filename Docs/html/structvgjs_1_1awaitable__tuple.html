Detailed Description

template<typename PT, typename... Ts>
struct vgjs::awaitable_tuple< PT, Ts >

Awaitable for scheduling jobs. All jobs are put into std::tuples. If one of the elements is a valid tag the jobs are scheduled for this tag and the coro is resumed immediately.
