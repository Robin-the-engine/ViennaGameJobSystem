Detailed Description

template<typename U>
struct vgjs::yield_awaiter< U >

When a coroutine calls co_yield this awaiter calls its parent.

A coroutine calling co_yield suspends with this awaiter. The awaiter is similar to the final awaiter, but always suspends.
